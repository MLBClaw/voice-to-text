# 🎙️ 语音转文字

一款基于 Web Speech API 的实时语音转文字应用，支持多语言识别、历史记录管理。

## ✨ 功能特性

### 🎤 核心功能
- **实时转写** - 边说边出文字，延迟低
- **多语言支持** - 中文、英文、日文、韩文、法文、德文、西班牙文、俄文
- **连续识别** - 长时间录音自动分段识别
- **自动标点** - 智能添加标点符号

### 📊 录音功能
- **录音计时** - 实时显示录音时长
- **音量可视化** - 动态音频波形显示
- **后台录音** - 切换应用后继续录音

### 💾 数据管理
- **历史记录** - 自动保存所有转写内容
- **本地存储** - 数据保存在设备本地
- **导出功能** - 支持导出为文本文件
- **分享功能** - 一键分享转写结果

### 📱 使用体验
- **PWA 支持** - 可添加到手机主屏幕，像原生 App 使用
- **响应式设计** - 完美适配手机和电脑
- **快捷操作** - 支持复制、分享、保存

## 🚀 使用方法

### 在线使用
访问：https://mlbclaw.github.io/voice-to-text

### 添加到手机主屏幕
1. 用 Chrome/Safari 打开应用
2. 点击菜单 → "添加到主屏幕"
3. 桌面上会出现 App 图标

### 使用步骤
1. 点击麦克风按钮开始录音
2. 对着设备说话
3. 文字实时显示在屏幕上
4. 点击停止按钮结束录音
5. 复制、分享或保存结果

## 🔒 隐私说明

- 所有转写过程在浏览器本地完成
- 数据仅存储在设备本地，不上传服务器
- 麦克风权限仅用于语音识别
